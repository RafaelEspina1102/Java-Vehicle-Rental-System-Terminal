
# Java Vehicle Rental System Terminal

A simple Terminal based Vehicle Rental System to learn more about Core Java

## Documentation

*This part is where I put my thought process on creating the Vehicle Rental System*

What do I need in a vehicle rental system?
- Customer
- Vehicle (Car, Motorcycle, Truck)
- Rental System
- Vehicle Manager

What does the 4 do and need?

**Customer**
- Name
- License Number
- Phone Number
- Rent a vehicle
- Return a vehicle

**Rental System**
- Customer
- Vehicle
- Date Rented
- Date Returned
- Cost

**Vehicle**
- Plate number
- Brand
- Model
- Year
- Availability

**Vehicle Manager**
- Add Vehicle
- Remove Vehicle
- Find Vehicle
- List Vehicle
